Таблица "customers" представляет информацию о клиентах. Она содержит следующие поля:

- id : уникальный идентификатор клиента.
- first_name : имя клиента.
- last_name : фамилия клиента.
- email : электронная почта клиента.
- address : адрес клиента.
- phone : номер телефона клиента.
- date_of_birth : дата рождения клиента.
- gender : пол клиента.
- loyalty_program_id : идентификатор программы лояльности, к которой привязан клиент.
- start_date : дата добавления клиента в систему.

Таблица "loyalty_programs" содержит информацию о программах лояльности. Она имеет следующие поля:

- id : уникальный идентификатор программы лояльности.
- program_name : название программы лояльности.
- discount : размер скидки, предоставляемой программой лояльности.

Таблица "products" содержит информацию о продуктах. Она включает следующие поля:

- id : уникальный идентификатор продукта.
- name : название продукта.
- price : цена продукта.
- description : описание продукта.
- supplier_id : идентификатор поставщика продукта.
- amount : количество продукта на складе.
- in_stock : флаг указывающий на наличие продукта на складе.

Таблица "orders" содержит информацию о заказах. Она включает следующие поля:

- id : уникальный идентификатор заказа.
- customer_id : идентификатор клиента, сделавшего заказ.
- paid_flg : флаг указывающий, оплачен ли заказ.
- order_date : дата заказа.

Таблица "order_details" содержит информацию о деталях заказа. Она включает следующие поля:

- id : уникальный идентификатор детали заказа.
- order_id : идентификатор заказа, к которому относится запись.
- product_id : идентификатор продукта, связанного с записью.
- product_count : количество продукта в детали заказа.


Таблица "suppliers" содержит информацию о поставщиках. Она включает следующие поля:

- id : уникальный идентификатор поставщика.
- name : имя поставщика.
- email : электронная почта поставщика.
- phone : номер телефона поставщика.
